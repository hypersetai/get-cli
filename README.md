# get-cli

Distribution repository for the **Hyperset CLI**.

Latest version: **v0.1.0**

## Install

### macOS / Linux

```bash
curl -fsSL https://hypersetai.com/cli/install.sh | bash
```

### Windows (PowerShell)

```powershell
irm https://hypersetai.com/cli/install.ps1 | iex
```

### npm

```bash
npm i -g @hypersetai/cli
```

### bun

```bash
bun add -g @hypersetai/cli
```
